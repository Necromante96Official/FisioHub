Termos de uso

Sistema local-first

FisioHub trabalha localmente no navegador e usa os dados importados apenas no dispositivo atual.

Uso dos dados

Os dados processados, backups e pendências ficam armazenados no navegador via localStorage, sem envio automático para serviços externos.

Responsabilidade

O usuário é responsável por manter backups válidos antes de limpar informações ou importar novas bases.

Atualizações

A versão exibida no rodapé é sincronizada automaticamente com o package.json e o ano é calculado pelo sistema no carregamento.
