Boas práticas

  • Confira a data antes de processar os dados.
  • Use backups antes de limpezas grandes.
  • Revise pacientes com nomes parecidos antes de confirmar conflitos.

Atenção

  • Limpar Texto remove apenas o rascunho importado.
  • Limpar Dados limpa dados das páginas, preservando a lista de pacientes.
  • Limpar Tudo deve ser usado apenas quando quiser reiniciar os dados locais.
